## 🚀 Projects & Research

### 🏆 [Transformer-based NIDS on Edge SoC](https://github.com/mukund01001/FlowTransformer_Quantization_Analysis)
* **DVCon India 2025 Design Contest Finalist (Top 25 out of 6000+ teams)** under the I²CS Research Lab.
* Architected a novel **tiled matrix multiplication accelerator in Verilog** to accelerate the critical QKV (Query, Key, Value) and FFN layers of a Transformer model on an FPGA.
* The design utilizes **pipelined Multiply-Accumulate (MCA) engines (II=1)**, loop unrolling, and partitioned BRAM buffers to maximize parallelism and data reuse.
* Designed a high-throughput **AXI-based SoC interface** with three parallel AXI4-FULL channels and a unified AXI-Lite control plane for matrix streaming and synchronization between the CPU and FPGA.
* The software component involved developing a quantized **DistilBERT-based NIDS model**, integrating it with the accelerator using C++ drivers, and creating a Python test script.
* Benchmarked a **7x speedup** over the software baseline and successfully processed real-time packet captures at 10 Gbps.
